How to instructions:

Step 1: Need to have a node.js installed on your machine ( https://nodejs.org/en/download/ )

Step 2: Install Gulp using NPM ( https://github.com/gulpjs/gulp/blob/master/docs/getting-started.md )

Step 3: Use NPM to install browser-sync ( $ npm install browser-sync gulp --save-dev )

Step 4: In Terminal at the project root directory type gulp serve

Step 5: Have a awesome time
